# Прогнозирование волатильности и торговые стратегии на российском фондовом рынке

Репозиторий к кандидатской диссертации **Лысенок Никиты Ильича**, НИУ ВШЭ.

---

## Структура репозитория

```
├── moex_discovery/       — Сбор и подготовка данных
├── moex_volatility/      — Модели и стратегии (главы 3–4)
└── dissertation_materials/
```

### `moex_discovery/` — Данные

Пайплайн загрузки, очистки и объединения данных из трёх источников:

- **MOEX ISS API** — 10-минутные свечи (OHLCV) по 17 тикерам голубых фишек
- **Yahoo Finance** — глобальные индексы (S&P 500, VIX, нефть Brent, золото)
- **Макропоказатели** — ключевая ставка ЦБ, CPI, ставка ФРС

Итоговый датасет: 17 тикеров, 2015–2025, реализованная волатильность (RV) из 10-минутных доходностей.

### `moex_volatility/` — Модели и стратегии

#### Глава 3. Прогнозирование реализованной волатильности

Семь семейств моделей прогнозирования RV на горизонтах 1, 5 и 22 дня:

| Категория | Модели |
|-----------|--------|
| Эконометрические | HAR-J, GARCH (GJR) |
| Градиентный бустинг | XGBoost, LightGBM |
| Нейросетевые | LSTM, GRU |
| Ансамбли | Адаптивные гибриды (HAR-J + лучший ML, 7 стратегий взвешивания) |

Оценка — walk-forward кросс-валидация (2017–2025, 17 тикеров), основная метрика — QLIKE. Статистические тесты: Diebold-Mariano, Mincer-Zarnowitz, Model Confidence Set.

Ключевые результаты и таблицы: `results/final/tables/`, `results/DISSERTATION_FINAL_RESULTS.md`.

#### Глава 4. Применение прогнозов волатильности в торговых стратегиях

Шесть стратегий трёх типов:

| Тип | Стратегии |
|-----|-----------|
| Контртренд | Mean Reversion, Bollinger Bands |
| Трендовые | Donchian Channel, Supertrend |
| Диапазонные | Pivot Points, VWAP |

Четыре подхода к использованию прогнозов волатильности:

- **A** — базовый (без прогнозов, walk-forward grid search)
- **B** — адаптивные стоп-лоссы (масштабирование по прогнозу σ)
- **C** — режимная фильтрация (торговля только в определённых режимах волатильности)
- **D** — vol-gating (размер позиции по прогнозу σ)

Walk-forward бэктест на дневных и часовых данных (2022–2025), 17 тикеров, с учётом реальных комиссий MOEX. EW-портфели и мета-портфели.

Ключевые результаты: `results/final/strategies/walkforward_v4/`, `FINAL_RESULTS.md`.
