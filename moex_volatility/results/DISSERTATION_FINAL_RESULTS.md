## 8. Нерешённые вопросы / Пробелы

### ❌ LSTM / GRU в Walk-Forward
Нейросетевые модели (LSTM, GRU) НЕ были успешно обучены в walk-forward режиме (`train_neural.log` показывает "Insufficient data" для всех тикеров). Результаты есть только для test 2019 (single-year). **Это пробел для Главы 3** — нужно либо:
- Переобучить с исправленным пайплайном
- Или объяснить, почему не включены в walk-forward (data requirements)

### ⚠️ Расхождение V3 vs V4 стратегий
V3 (gross, 0 bps) и V4 (commission-optimized, 40 bps) дают разные абсолютные числа. Median Sharpe в V3 gross = 0.45–1.0, в V4 net = 0.14–0.70. Это ожидаемо (V4 строже), но нужно чётко указать в диссертации, какие числа из какого пайплайна.

### ⚠️ S5/S6 в post-hoc анализе (commission_moex_real.txt)
S5_PivotPoints и S6_VWAP в post-hoc используют СТАРЫЕ версии (до улучшений s5_rerun/s5s6_rerun). В V4 pipeline эти стратегии уже улучшены.

### ⚠️ Approaches E/F/G
В `walkforward/` (V3 gross) есть результаты для подходов E, F, G (расширенные комбинации). В финальном V4 — только A, B, C, D. E/F/G = комбинации B+C, C+D, B+C+D соответственно. Они есть gross, но не net.

### ⚠️ Статистическая значимость мета-портфелей
Bootstrap CI для META-A vs META-BEST = [0.82, 1.68] (не включает 0), но t-test p=0.254. Это из-за малого числа OOS лет (4). Нужно обсудить limitations.

### ✅ GARCH в диссертации
GARCH результаты есть для test 2019 (log-QLIKE: -7.44 H=1, -6.15 H=5, -4.25 H=22). GARCH значительно хуже HAR-J и ML моделей на H=5 и H=22 (многошаговые прогнозы сходятся к безусловной дисперсии).
