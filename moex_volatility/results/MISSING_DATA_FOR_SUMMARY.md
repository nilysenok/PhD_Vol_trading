## 1. Формула реализованной волатильности (RV)

**Источник**: `src/data/intraday_features.py` (строки 95–203)

### Базовая формула

$$RV_t = \sum_{i=1}^{N} r_{t,i}^2$$

где $r_{t,i} = \ln(C_{t,i} / C_{t,i-1})$ — логарифмическая доходность $i$-го 10-минутного бара дня $t$, $N$ — число баров за торговую сессию (10:00–18:50 МСК, ~53 бара).

### Дополнительные компоненты

| Мера | Формула | Описание |
|------|---------|----------|
| RV | $\sum r_i^2$ | Реализованная дисперсия |
| RSV⁺ | $\sum r_i^2 \cdot \mathbb{1}(r_i > 0)$ | Положительная полу-дисперсия |
| RSV⁻ | $\sum r_i^2 \cdot \mathbb{1}(r_i < 0)$ | Отрицательная полу-дисперсия |
| BV | $\frac{\pi}{2} \sum |r_i| \cdot |r_{i-1}|$ | Bipower Variation |
| Jump | $\max(RV - BV, 0)$ | Компонента скачков |
| RSkew | $\frac{\sum r_i^3}{RV^{3/2}}$ | Реализованная асимметрия |
| RKurt | $\frac{\sum r_i^4}{RV^2}$ | Реализованный эксцесс |
| RV_morning | $\sum r_i^2$ (10:00–12:00) | Утренняя волатильность |
| RV_midday | $\sum r_i^2$ (12:00–16:00) | Дневная волатильность |
| RV_evening | $\sum r_i^2$ (16:00–18:50) | Вечерняя волатильность |

### Аннуализация

$$\sigma_{ann} = \sqrt{RV} \times \sqrt{252}$$

### HAR-агрегации (features.py, строки 104–140)

| Переменная | Формула | Смысл |
|-----------|---------|-------|
| rv_d | $RV_{t-1}$ | Дневное RV (лаг 1) |
| rv_w | $\frac{1}{5}\sum_{i=1}^{5} RV_{t-i}$ | Среднее за 5 дней |
| rv_2w | $\frac{1}{10}\sum_{i=1}^{10} RV_{t-i}$ | Среднее за 10 дней |
| rv_m | $\frac{1}{22}\sum_{i=1}^{22} RV_{t-i}$ | Среднее за 22 дня |
| rv_q | $\frac{1}{66}\sum_{i=1}^{66} RV_{t-i}$ | Среднее за 66 дней |

### Целевая переменная

$$y_{t,h} = RV_{t+h}$$

- H=1: прогноз RV на 1 день вперёд (`rv.shift(-1)`)
- H=5: прогноз RV на 5 дней вперёд (`rv.shift(-5)`)
- H=22: прогноз RV на 22 дня вперёд (`rv.shift(-22)`)

**Важно**: целевая переменная — это RV **одного дня** через $h$ дней, а не сумма/среднее RV за $h$ дней.

## 7. DM-тесты подходов (META-уровень)

**Источник**: `results/final/strategies/walkforward_v4/tables/stat_tests.csv`

### 7.1 META-портфели (A vs forecast approaches)

| Сравнение | Δ Sharpe | DM t-stat | DM p-value | Bootstrap 95% CI | Значим? |
|-----------|---------|-----------|------------|-----------------|---------|
| META-D vs META-A | +1.13 | 0.944 | 0.345 | [0.76, 1.59] excl 0 | Bootstrap: **Да** |
| META-BEST vs META-A | +1.19 | 1.14 | 0.254 | [0.82, 1.68] excl 0 | Bootstrap: **Да** |

**Интерпретация расхождения:**
- t-test/DM проверяет H0: mean(r_BEST − r_A) = 0 — разницу **доходностей**
- Bootstrap строит CI для разницы **Sharpe** = mean/vol
- META-BEST улучшает прежде всего через снижение волатильности (2.79% vs 4.37%), а не рост доходности
- Поэтому t-test незначим (шум доходностей), а Bootstrap значим (стабильное улучшение Sharpe)

## 12. Объяснение A vs BCD периодов

**Источник**: `scripts/strategies_walkforward.py`, строки 72–73, 1750–1925

### Определение

```python
A_TEST_YEARS = list(range(2020, 2027))      # 2020–2026
BCD_TEST_YEARS = list(range(2022, 2027))    # 2022–2026
```

### Причина различия

**Подход A** (2020–2026):
- Не использует прогнозы волатильности
- Walk-forward: расширяющееся окно (все данные до test_year)
- Grid search параметров стратегии на train-данных
- Может начинать OOS с 2020, т.к. не нуждается в val-периоде для калибровки прогнозов

**Подходы B, C, D** (2022–2026):
- Используют прогнозы σ_pred (из walk-forward моделей волатильности)
- Требуют validation-период для калибровки prediction-dependent параметров:
  - B: пороги адаптивных стопов
  - C: режимные фильтры
  - D: пороги vol-gate
- Val-период: 2020–(test_year−1)
- Минимум 2 года val (2020–2021) для надёжной калибровки → OOS начинается с 2022

### Код (из strategies_walkforward.py)

```python
# Approach A: expanding train_mask
train_mask[WARMUP:train_end] = True  # WARMUP=200 bars

# Approach B: validation from 2020
_, val_start = _year_bounds(dates, 2020)     # val starts at 2020
_, val_end = _year_bounds(dates, test_year)   # val ends before test year
val_mask[val_start:val_end] = True
```

### Следствие для итоговых таблиц

- **META-A** считается за 2020–2026 (7 лет OOS)
- **META-B/C/D/BEST** считаются за 2022–2026 (5 лет OOS)
- Для корректного сравнения в итоговых таблицах все META-портфели пересчитаны на **общий период 2022–2026** (файл `meta_portfolios_bcd.csv`)
