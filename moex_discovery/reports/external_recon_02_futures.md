# Разведка фьючерсов MOEX ISS

**Дата:** 2026-02-05
**Целевой период:** 2014-08-26 — 2026-02-03

## КРИТИЧЕСКОЕ ОТКРЫТИЕ

### ❌ Фьючерсы Si/BR/RI — НЕ ПОДХОДЯТ для нашего периода

| Фьючерс | Тип | 10-мин данные с | Проблема |
|---------|-----|-----------------|----------|
| **Si** (USD/RUB) | Квартальный | 2024-03-15 | Только ~2 года данных |
| **BR** (Brent) | Ежемесячный | ~2023 | Только ~3 года данных |
| **RI** (RTS index) | Квартальный | ~2023 | Только ~3 года данных |
| **GD** (Gold) | Квартальный | ~2024 | Только ~2 года данных |
| **Eu** (EUR/RUB) | Квартальный | ~2024 | Только ~2 года данных |

**Причина:** MOEX ISS API не содержит исторических 10-минутных свечей для фьючерсов до 2022-2023 года. Старые данные не были оцифрованы в candles API.

## Структура фьючерсных контрактов

### Si (USD/RUB) — квартальный
- Коды месяцев: H(мар), M(июн), U(сен), Z(дек)
- Формат: Si{M}{Y} — например SiH6 = март 2026
- Контрактов за период 2014-2026: ~48

### BR (Brent) — ежемесячный
- Коды месяцев: F G H J K M N Q U V X Z (все 12)
- Формат: BR{M}{Y} — например BRF6 = январь 2026
- Контрактов за период 2014-2026: ~144

### RI (RTS index) — квартальный
- Аналогично Si
- Контрактов: ~48

## Альтернативы

### ✅ USD/RUB SPOT (USD000UTSTOM) — ПОДХОДИТ!

| Параметр | Значение |
|----------|----------|
| Секция | Валютная (currency/selt) |
| Board | CETS |
| 10-мин с | **2011-12-08** |
| Баров/день | 73-74 |
| Покрытие | 2014-08-26 ✓ но обрывается 2024-06-11 ⚠ |

**API:**
```
https://iss.moex.com/iss/engines/currency/markets/selt/boards/CETS/securities/USD000UTSTOM/candles.json?interval=10&from=DATE&till=DATE
```

**Примечание:** Данные обрываются в июне 2024. Возможно, ликвидность перенесена на другой инструмент или изменилась структура торгов.

### ✅ RVI (Индекс волатильности) — ПОДХОДИТ!

- Implied volatility российского рынка
- 10-мин данные с 2013-12-16
- Полностью покрывает наш период
- 89 баров в день

## Выводы

1. **Фьючерсы не подходят** для нашего исследования — нет исторических данных до 2022 года в ISS API

2. **Для анализа валютной волатильности:**
   - USD000UTSTOM: 2014-08-26 — 2024-06-11 (неполное покрытие)
   - Нужно искать другие источники для периода после июня 2024

3. **Для бенчмарка волатильности:**
   - RVI — лучший выбор (implied volatility)
   - IMOEX — можно рассчитать realized volatility индекса

4. **Нет готовых непрерывных рядов** фьючерсов в ISS API

## Рекомендации для исследования

1. ✅ Скачать **RVI** как бенчмарк implied volatility
2. ✅ Скачать **IMOEX** для расчёта RV индекса
3. ⚠️ **USD000UTSTOM** можно использовать для периода 2014-2024, но с осторожностью
4. ❌ **Фьючерсы** — не использовать из-за отсутствия исторических данных
